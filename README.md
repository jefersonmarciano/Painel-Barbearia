## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DATABASE_URL="file:./dev.db"`

`AWS_ACCESS_KEY_ID`

`AWS_SECRET_ACCESS_KEY`

`AWS_REGION`

`ACCESS_KEY_TOKEN`

## Deployment

Para executar o projeto:

```bash
  yarn install
```


```bash
  yarn dev
```
